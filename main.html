Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in the are getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML content,innerText gets/sets visible text ,textContent gets/sets all text.

Question-3: What is event delegation in the DOM?

Event delegation allows us to attach a single event listener to a parent element, which handles events from its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling means an event starts from the target element and moves up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tag'), Add: parent.appendChild(element) or parent.append(element), Remove: parent.removeChild(element) or element.remove().
